## üì• Download & Installation

You can download the latest version of the extension directly from our GitHub Releases page.

1. **Download**: Go to the [**Releases Tab**](../../releases) and download the `Form-Automator-v1.0.zip` file.
2. **Unzip**: Extract the contents of the ZIP file to a folder on your computer (e.g., `Documents/Form-Automator`).
3. **Open Chrome Extensions**:
   - Open Google Chrome.
   - Navigate to `chrome://extensions/` in the address bar.
4. **Enable Developer Mode**:
   - Toggle the **"Developer mode"** switch in the top right corner.
5. **Load Extension**:
   - Click the **"Load unpacked"** button.
   - Select the folder where you extracted the files.

üéâ **That's it! The extension is now installed and ready to use.**

## üõ†Ô∏è How to Use

### Method 1: The Popup
1. Navigate to your maintenance form.
2. Click the **Form Automator icon** in your browser toolbar.
3. Click the **"Fill Form Now"** button.

### Method 2: The Shortcut
1. On any page, simply press the **Backtick key (`` ` ``)** (usually below `Esc`).
2. The form gets filled instantly!

### Method 3: Auto-Fill
1. Open the extension popup.
2. Toggle the **"Auto-fill on load"** switch.
3. Now, any page you visit will be filled automatically as soon as it loads.
