# Form Automator - Maintenance Helper

A simple Chrome extension designed to help fill repetitive maintenance forms.

## Features
- Writes **"ok"** into all text boxes and textareas.
- Checks **every checkbox** on the page.
- Beautiful, modern UI for easy triggering.

## How to Install
1. Open Google Chrome.
2. Go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the folder containing these files.

## How to Use
1. Navigate to the form you want to fill.
2. Click the **Form Automator** icon in your extension bar.
3. Click the **Fill Form Now** button.
